USAGE STEPS (Desktop GUI):
=========================

1. Launch the application:
   python sudoku_gui.py

2. Select a puzzle difficulty:
   - Easy (0 backtracks)
   - Hard (14 backtracks)
   - Very Hard (49 backtracks)

3. Click "Load Puzzle":
   - Grid displays with blue numbers (given clues)

4. Click "Solve Puzzle":
   - Algorithm runs
   - Green numbers appear (solved cells)
   - Statistics update

5. View results:
   - Check statistics panel
   - See difficulty level
   - View backtrack information

6. Reset with "Reset Puzzle" button:
   - Returns to original puzzle

CUSTOMIZATION:
==============

Desktop GUI (sudoku_gui.py):
1. Colors: Modify self.BG_COLOR, self.FRAME_COLOR, etc.
2. Font: Change tkFont specifications
3. Layout: Modify grid_frame and control_panel layouts
4. Buttons: Adjust button properties

Web UI (style.css):
1. Colors: Modify :root CSS variables
2. Animations: Edit @keyframes definitions
3. Layout: Adjust grid layouts and spacing
4. Responsive: Modify media query breakpoints

TROUBLESHOOTING:
================

Desktop GUI Issues:
1. Tkinter not found:
   - Windows: Should be included
   - Ubuntu: sudo apt-get install python3-tk
   - Mac: brew install python-tk

2. Puzzle files not found:
   - Ensure easy.txt, hard.txt, veryhard.txt in working directory
   - Check file paths

Web UI Issues:
1. Port 5000 already in use:
   - Change port in sudoku_web_app.py line 55
   - python sudoku_web_app.py --port 5001

2. Flask not installed:
   - pip install flask
   - pip install flask-cors (if needed)

3. Templates not found:
   - Ensure templates/ folder exists
   - Check templates/index.html exists
