How to Use

  1. Load a Puzzle: Select difficulty level and click "Load Puzzle"
  2. View Grid: The puzzle will display with given numbers in blue
  3. Solve: Click "Solve Puzzle" to use the CSP algorithm
  4. View Results: Solved numbers appear in green
  5. Check Stats: See backtracking statistics and difficulty level
  6. Reset: Click "Reset" to load the original puzzle

Algorithm Details

This solver uses a Constraint Satisfaction Problem (CSP) approach with:

  • AC-3: Maintains arc consistency between cells
  • Backtracking: Searches for valid assignments
  • Forward Checking: Detects failures early
  • MRV: Prioritizes most constrained cells
