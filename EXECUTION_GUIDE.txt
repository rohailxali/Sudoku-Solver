ALGORITHM COMPONENTS:
=====================

1. AC-3 Constraint Propagation
   - Enforces arc consistency between neighboring cells
   - Reduces domain of each cell based on assigned neighbors
   - Time: O(e * d²) where e=arcs, d=domain size

2. Naked Singles
   - Identifies cells with only one possible value
   - Propagates these forced assignments
   - Iterates until convergence

3. Forward Checking
   - After each assignment, removes value from neighbors
   - Detects contradictions early
   - Prevents deep recursion into invalid branches

4. MRV Heuristic
   - Selects variable with minimum remaining values first
   - Prioritizes most constrained cells
   - Fails fast on poor choices

5. Backtracking Search
   - Depth-first search with chronological backtracking
   - Only invoked when constraint propagation insufficient
   - Recursively explores value assignments

KEY INSIGHTS:
=============

1. Constraint propagation is extremely powerful
   - Easy puzzle solved without any backtracking
   - Reduces search space from 9^81 to ~0 nodes

2. AC-3 efficiency critical
   - Pre-computed neighbors avoid redundant calculations
   - Result: O(n²) constraint checking instead of O(n³)

3. MRV heuristic effectiveness
   - Hard puzzle: only 14 backtracks vs hundreds/thousands
   - Very hard puzzle: only 49 backtracks - still tractable

4. Failure ratios reveal puzzle nature
   - Easy: 0% (no backtracking needed)
   - Hard: 50% (good heuristic - balanced puzzle)
   - Very Hard: 181% (complex structure - many wrong turns)

TROUBLESHOOTING:
================

If puzzles don't solve:
1. Verify input files have exactly 9 lines and 9 digits per line
2. Check that 0 represents empty cells (1-9 are givens)
3. Ensure puzzle is valid (no duplicate givens)
4. Run with fresh Python interpreter

If output doesn't display:
1. Check terminal encoding is UTF-8 compatible
2. Special characters (| and -) require Unicode support
3. Use the solution directly if display issues occur
