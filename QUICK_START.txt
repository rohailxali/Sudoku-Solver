Two beautiful UIs are now available for the Sudoku Solver!

OPTION 1: DESKTOP GUI (Tkinter) ⭐ Recommended for Local Use

QUICKEST START:
1. Open PowerShell in c:\Docs\AI
2. Run: python sudoku_gui.py
3. Window opens automatically!

FEATURES:
✓ Dark theme with professional styling
✓ Beautiful 9x9 grid with 3x3 box borders
✓ Color-coded numbers (blue = given, green = solved)
✓ Load puzzles: Easy, Hard, Very Hard

✓ Status updates
✓ Modern buttons with hover effects

- Python 3.7+ (Tkinter built-in)

SETUP:

STEP-BY-STEP TUTORIALS
═══════════════════════════════════════════════════════════════

DESKTOP GUI TUTORIAL:
=====================

Step 1: Open Command Prompt
  - Press Win+R
  - Type: cmd
  - Navigate to: cd c:\Docs\AI

Step 2: Launch GUI
  - Type: python sudoku_gui.py
  - Press Enter
  - Window opens (wait 1-2 seconds)

Step 3: Load Easy Puzzle
  - Verify "easy.txt" is selected
  - Click "Load Puzzle" button
  - Board displays with blue numbers

Step 4: Solve It
  - Click "Solve Puzzle"
  - Green numbers fill in the empty cells
  - Statistics update automatically

Step 5: View Results
  - Check the statistics panel on the right:
    * BACKTRACK calls: 0
    * Failure count: 0
    * Difficulty: Easy

Step 6: Try Another Puzzle
  - Select "hard.txt" or "veryhard.txt"
  - Click "Load Puzzle"
  - Click "Solve Puzzle" again
  - Compare statistics!


WEB UI TUTORIAL:
================

Step 1: Install Flask (One Time)
  - Open PowerShell
  - Type: pip install flask
  - Wait for installation

Step 2: Start Web Server
  - Navigate: cd c:\Docs\AI
  - Type: python sudoku_web_app.py
  - You see: "Running on http://localhost:5000"

Step 3: Open Browser
  - Open Chrome, Firefox, or Edge
  - Go to: http://localhost:5000
  - Beautiful page loads!

Step 4: Load Puzzle
  - Select difficulty (Easy/Hard/Very Hard)
  - Click "Load Puzzle" button
  - Grid appears with blue numbers

Step 5: Solve
  - Click "Solve Puzzle"
  - Green numbers populate
  - Statistics update live

Step 6: View Help
  - Click "Help" button
  - Modal explains everything
  - Press Escape to close

Step 7: Try More Puzzles
  - Load different puzzles
  - Note the difference in backtrack counts
  - Easy = 0, Hard = 14, Very Hard = 49

═══════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

DESKTOP GUI WON'T START:
Problem: "No module named 'tkinter'"
Solution: 
  - Windows: Usually built-in
  - Mac: brew install python-tk
  - Linux: sudo apt-get install python3-tk

WEB UI WON'T START:
Problem: "No module named 'flask'"
Solution: 
  pip install flask

PUZZLE FILES NOT FOUND:
Problem: "FileNotFoundError: easy.txt not found"
Solution:
  - Check you're in c:\Docs\AI directory
  - Verify easy.txt, hard.txt, veryhard.txt exist
  - Copy if missing from assignment files

PORT 5000 ALREADY IN USE:
Problem: "Address already in use"
Solution:
  - Close other Flask apps
  - OR change port in sudoku_web_app.py line 55

BROWSER NOT LOADING:
Problem: Page won't open at localhost:5000
Solution:
  - Check Flask server is running
  - Refresh the page (Ctrl+R)
  - Try different browser
  - Restart Flask server
