TROUBLESHOOTING
───────────────

PROBLEM: "Authentication failed"
SOLUTION:
  - GitHub requires Personal Access Token for HTTPS
  - Go to: https://github.com/settings/tokens
  - Click "Generate new token"
  - Select scopes: repo (all options)
  - Copy token and use as password when prompted

PROBLEM: Already have SSH key?
SOLUTION:
  Use SSH instead of HTTPS:
  git remote add origin git@github.com:rohailxali/YOUR-REPO-NAME.git
  git push -u origin main

PROBLEM: "Remote already exists"
SOLUTION:
  Remove existing remote first:
  git remote remove origin
  Then add again with correct URL
