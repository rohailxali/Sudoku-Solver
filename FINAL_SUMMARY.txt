1. AC-3 ARC CONSISTENCY ALGORITHM
   - Maintains constraint consistency between variables
   - Reduces domain sizes iteratively
   - Key function: ac3()

2. BACKTRACKING SEARCH WITH FORWARD CHECKING
   - Intelligent search using domain reduction
   - Prunes impossible branches early
   - Key function: backtrack()

3. MINIMUM REMAINING VALUES (MRV) HEURISTIC
   - Selects variable with smallest domain first
   - Reduces search space exponentially
   - Integrated throughout backtracking

4. NAKED SINGLES CONSTRAINT PROPAGATION
   - Detects forced assignments (1 value left)
   - Additional domain pruning technique
   - Complements AC-3 for efficiency

5. FORWARD CHECKING
   - Immediate domain reduction after assignment
   - Detects conflicts early
   - Prevents wasted search effort

⚡ OPTIMIZATION TECHNIQUES
────────────────────────────

1. Pre-computed Neighbors
   - All cell neighbors pre-calculated at init
   - Avoids repeated computation during search
   - Significant speed improvement

2. Domain Representation
   - Sets used for domains (O(1) lookups)
   - Fast membership testing
   - Efficient removal operations

3. Constraint Propagation Priority
   - AC-3 runs first (general constraints)
   - Naked singles as secondary (specific cases)
   - Maximizes early pruning

4. MRV Heuristic
   - Choose most constrained variable first
   - Reduces branching factor
   - Exponential search space reduction

5. Early Termination
   - Stops immediately when unsolvable
   - Avoids exploring dead-end branches
   - Contributes to fast failure detection


💡 HOW THE SOLVER WORKS (Simple Explanation)
───────────────────────────────────────────────

Step 1: SETUP
- Start with puzzle (numbers given, blanks empty)
- Each cell can have values 1-9 (initially)
- Apply Sudoku rules to narrow possibilities

Step 2: CONSTRAINT PROPAGATION
- AC-3 algorithm removes impossible values
- Whenever a cell is assigned, eliminate that value from neighbors
- Repeat until no more reductions possible
- (Easy puzzles often solve here with 0 backtracks!)

Step 3: NAKED SINGLES
- Look for cells with only 1 possible value
- Assign it automatically (no guessing needed)
- Continue propagating

Step 4: BACKTRACKING (if needed)
- If propagation alone doesn't solve it
- Pick least constrained cell (MRV heuristic)
- Try one possibility
- Recursively solve rest
- If stuck, backtrack and try different value

Step 5: SOLUTION
- Continue until all cells filled
- Verify no Sudoku rules violated
- Return complete solution
