📝 GITHUB AUTHENTICATION TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

If you get "Authentication failed":

1. GitHub moved to Personal Access Tokens (PAT)
   
2. Create a token:
   • Go to https://github.com/settings/tokens/new
   • Log in if needed
   • Click "Generate new token (classic)"
   • Name it: "git-push"
   • Select scope: repo (all options under it)
   • Generate token
   • Copy the token (long string of characters)
   
3. When git asks for password, pasta the token (not your password)

4. Git will remember it for future pushes
