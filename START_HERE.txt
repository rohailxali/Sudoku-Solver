🎓 HOW EVERYTHING WORKS (ULTRA-QUICK VERSION)
═════════════════════════════════════════════════════════════════════════════

The Algorithm (AC-3 + Backtracking):

1. AC-3 CONSTRAINT PROPAGATION
   ├─ Start: Each cell could be 1-9
   ├─ Observation: If cell A=5, no neighbor can be 5
   ├─ Action: Remove 5 from all neighbors
   ├─ Result: Domains get smaller
   └─ Easy puzzles: Solves 100% of the time with this alone!

2. NAKED SINGLES (Forced Assignments)
   ├─ Find: Cells with only 1 possible value
   ├─ Assign: That value automatically
   ├─ Propagate: Remove from neighbors
   └─ Chain Reaction: Keeps happening until done

3. BACKTRACKING (If needed)
   ├─ Select: Most constrained unassigned cell (MRV)
   ├─ Try: Each value in its domain
   ├─ Recurse: Continue solving
   ├─ Backtrack: If stuck, undo and try next value
   └─ Hard puzzles: Completes quickly despite needing this
